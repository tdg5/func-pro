Functional C#

C#, functional since 3.0

  • First-class functions
  • Lambda expressions
  • Support for immutability via the readonly keyword
    • Allows developer to embrace immutability where helpful
  • Linq
    • Declarative queries instead of step-by-step imperative process

// Lambda expression:
string[] words = { "cherry", "apple", "blueberry" };
int shortestWordLength = words.Min(w => w.Length);

// readonly keyword:
class Age {
  readonly int _year;
  Age(int year) { _year = year; }

// Linq:

                                where num < 3 || num > 7
                                select num;
