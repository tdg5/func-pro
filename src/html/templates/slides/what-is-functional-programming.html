What is functional programming?

func·tion·al pro·gram·ming

/fŭngk′shə-nəl prō′grăm′ĭng/

noun

A declarative programming paradigm that treats computation as the evaluation of mathematical functions and avoids changing-state and mutable data.

Huh?

"Functional programming combines the flexibility and power of abstract mathematics with the intuitive clarity of abstract mathematics."

What is functional programming?

  • A set of assumptions, concepts, values, and practices that constitutes one perspective on computer programming.
  • Though some languages take the functional paradigm to an extreme, functional programming has less to do with the specific language and more to do with the concepts and constructs used.
    • Many "mostly imperative" programming languages include functional concepts.
